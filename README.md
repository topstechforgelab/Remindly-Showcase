
⏰ Remindly
A simple and modern reminder management application built with C# and SQLite to help users organize tasks, schedules, and important events efficiently.

🧩 Overview
Remindly is a desktop productivity application designed to create, manage, and track reminders in a clean and user-friendly interface. It provides a lightweight solution for personal task management while storing data locally using SQLite.

✨ Features
📅 Create and manage reminders
🔔 Scheduled reminder notifications
🔍 Search and filter reminders
✏️ Edit and update existing reminders
🗑️ Delete completed or unwanted reminders
💾 Local SQLite database storage
🎨 Clean and intuitive desktop interface
🧰 Built With
Language: C# (.NET)
Framework: WPF
Database: SQLite
IDE: Visual Studio 2022
Technologies: MVVM Pattern, Microsoft.Data.Sqlite
📂 Project Structure
Models – Reminder data models
Services – Database and reminder services
Views – Application windows and pages
ViewModels – UI logic and data binding
Data – SQLite database files and configuration
🚀 Future Enhancements
🔄 Recurring reminders
📧 Email notifications
☁️ Cloud synchronization
🏷️ Categories and tags
🌙 Dark mode support
📱 Cross-platform support
